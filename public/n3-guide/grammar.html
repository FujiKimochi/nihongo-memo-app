N3 文法詳細解說

首頁 文法解說 單字大辭典
1. ～うちに
使用時機 1. 趁著某個狀態還在持續的時候，做某件事（否則之後就難以進行）。
2. 在做某件事的過程中，不知不覺發生了某種變化。
接續方式 動詞字典形 / ている形 / ない形 + うちに
い形容詞 + い + うちに
な形容詞 + な + うちに
名詞 + の + うちに
詳細例句解說
例句
動詞日本にほんにいるうちに、一度富士山ふじさんに登のぼってみたい。
想趁著人在日本的時候，去爬一次富士山。
動詞(否定)忘わすれないうちに、メモを取記っておこう。
趁還沒忘記，趕快把筆記記下來。
い形容詞熱あついうちに、どうぞ召めし上がってください。
請趁熱吃吧。
な形容詞元気げんきなうちに、いろいろな国くにへ旅行りょこうしたい。
趁身體還健康的時候，想去各個國家旅行。
名詞若わかいうちに、たくさん勉強べんきょうしておきなさい。
趁年輕的時候，要多讀點書。
2. ～ようとする
使用時機 正要、打算做某個動作時；或試圖做某事。
接續方式 動詞意向形 + と模式（とする / としている / とした）
詳細例句解說
動詞意向形出でかけようとしたとき、雨あめが降ふってきました。
正打算出門的時候，下起雨來了。
動詞意向形電車でんしゃに乗のろうとしたが、ドアが閉しまってしまった。
想要搭上電車，但門卻關上了。
3. ～ことがある
使用時機 有時會、偶爾會發生某事。
接續方式 動詞字典形 / ない形 + ことがある
詳細例句解說
字典形たまに自分じぶんで料理りょうりをすることがあります。
偶爾會自己做菜。
ない形朝食ちょうしょくを食たべないことがあります。
有時候會不吃早餐。
4. ～ばかり
使用時機 1. 剛做完某事。
2. 盡是、只管。
接續方式 動詞た形 + ばかり / 名詞 + ばかり
詳細例句解說
動詞た形さっき昼ひるごはんを食たべたばかりなのに、もうおなかが空すいた。
明明剛吃過午飯，卻已經餓了。
名詞彼はゲームばかりしている。
他整天盡是在玩遊戲。
5. ～たところ

接續方式 動詞た形 + ところ
詳細例句解說
