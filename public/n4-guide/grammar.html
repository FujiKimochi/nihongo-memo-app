N4 文法詳細解說

首頁 文法解說 單字大辭典
1. ～あげく
用法：表示經過了很長一段時間，最後卻得到了一個不理想的結果。常譯為「最後...」。
接續：た形 / 名詞+の + あげく
詳細例句
散々さんざん悩なやんだあげく、辭めることにした。
煩惱了很久，最後決定辭職。
2. ～あいだ / ～あいだに
用法：「あいだ」表示在某個期間內一直持續某動作；「あいだに」表示在某段期間內的某個時刻發生了某個動作。常譯為「在...期間」。
接續：動詞字典形 / ている形 / 普通形 / 名詞+の + あいだ(內)
詳細例句
夏休なつやすみのあいだ、ずっと國くににいました。
暑假期間，一直待在國內。
母ははが寝ねているあいだに、宿題しゅくだいをしました。
趁著母親睡覺的時候，做了作業。
子供こどもが学校がっこうにいっているあいだ、掃除そうじをします。
趁孩子去上學的時候打掃。
3. ～あまり～ない
用法：表示程度不高。常譯為「不太...」、「不怎麼...」。
接續：あまり + 動詞否定形 / 形容詞否定形
詳細例句
刺身さしみはあまり好すきではありません。
不太喜歡生魚片。
最近さいきんあまり本ほんを読よみません。
最近不怎麼讀書。
この料理りょうりはあまり辛からくないです。
這道菜不太辣。
4. ～ほうがいい
用法：用於給予建議或忠告。常譯為「還是...比較好」。
接續：動詞た形 / ない形 + ほうがいい
詳細例句
早はやく寝ねたほうがいいですよ。
還是早點睡覺比較好喔。
甘あまいものをあまり食たべないほうがいいです。
還是不要吃太多甜食比較好。
病院びょういんへ行いったほうがいいですよ。
還是去醫院看病比較好喔。
5. ～かもしれない
用法：表示可能性較低（約50%以下）的推測。常譯為「也許...」、「可能...」。
接續：普通形 / 名詞 / な形容詞 (去だ) + かもしれない
詳細例句
午後ごごから雨あめが降ふるかもしれません。
下午可能會下雨。
彼はもう帰かえったかもしれません。
他可能已經回去了。
あの店みせは有名ゆうめいかもしれません。
那家店也許很有名。
6. ～かどうか
用法：將「是否...」這個問句放入一個長句中時使用。常譯為「是否...」。
接續：普通形 / 名詞 / な形容詞 (去だ) + かどうか
詳細例句
明日あした雨あめが降ふるかどうか知しりません。
不知道明天是否會下雨。
この料理りょうりが美味おいしいかどうか食たべてみます。
這道菜好不好吃，我吃吃看。
彼かれが来くるかどうか、聞きいてください。
請問一下他是否會來。
7. ～てみる
用法：表示嘗試做某件事。常譯為「做做看...」、「試著...」。
接續：動詞て形 + みる
詳細例句
新あたらしい料理りょうりを作つくってみました。
試著做了新的料理。
このズボンを穿はいてみてもいいですか。
可以試穿看看這件褲子嗎？
分わからないので、先生せんせいに聞きいてみます。
因為不懂，所以去問問看老師。
8. ～てしまう
用法：1.表示動作完全完成。 2.表示對某個不可挽回的結果感到遺憾。常譯為「...完了」、「...掉了」。
接續：動詞て形 + しまう
詳細例句
宿題しゅくだいを全部やってしまいました。
作業全部都做完了。
財布さいふを忘わスれてしまいました。
不小心忘記帶錢包了（遺憾）。
電車でんしゃの中に傘かさを忘わスれてしまいました。
不小心把傘忘在電車上了（遺憾）。
9. ～ことができる
用法：表示能力或可能性。常譯為「會...」、「能夠...」。
接續：動詞字典形 + ことができる
詳細例句
日本語にほんごを話はなすことができます。
我會說日語。
ここで写真しゃしんを撮とることができます。
這裡可以拍照。
漢字かんじを書かくことができません。
我不會寫漢字。
10. ～はずだ
用法：基於某種理由或客觀情況進行的有把握推斷。常譯為「應該會...」、「照理說...」。
接續：普通形 / 名詞+の / な形容詞+な + はずだ
詳細例句
彼かれは今日来くるはずです。
他今天應該會來。
この本ほんは面白おもしろいはずです。
這本書理應很有趣。
田中たなかさんは病気びょうきのはずです。
田中先生應該是生病了。
11. ～までに
用法：表示截止期限，即在某個時間點之前必須完成動作。常譯為「在...之前」。
接續：名詞 / 動詞字典形 + までに
詳細例句
５時じまでに帰かえってください。
請在五點之前回來。
会議かいぎが始はじまるまでに、準備じゅんびをします。
在會議開始之前，先做好準備。
来週らいしゅうの月曜日げつようびまでに、レポートを出してください。
請在下週一之前交報告。
12. ～ながら
用法：表示同時進行兩個動作，重點在後面的動作。常譯為「一邊...一邊...」。
接續：動詞ます形(去ます) + ながら
詳細例句
音楽おんがくを聴ききながら、勉強べんきょうします。
一邊聽音樂一邊讀書。
歩あるきながら、スマホをしないでください。
請不要邊走路邊用手機。
歌うたを歌うたいながら、料理りょうりをします。
一邊唱歌一邊做菜。
13. ～ことがある（經驗）
用法：表示過去曾有過的經歷或經驗。常譯為「曾經...過」。
接續：動詞た形 + ことがある
詳細例句
日本にほんへ行いったことがあります。
我曾經去過日本。
富士山ふじさんに登のぼったことがありますか。
你有爬過富士山嗎？
納豆なっとうを食たべたことがありません。
我從來沒吃過納豆。
14. ～ことがある（有時）
用法：表示有時候會發生某種情況。常譯為「有時會...」。
接續：動詞字典形 / ない形 + ことがある
詳細例句
朝あさごはんを食たべないことがあります。
有時候會不吃早餐。
たまに会社かいしゃを休やすむことがあります。
偶爾也會有請假不去公司的時候。
夜よる、なかなか寝ねられないことがあります。
有時候晚上會怎麼也睡不著。
15. ～つもりだ
用法：表示說話人的計畫或打算。常譯為「打算...」。
接續：動詞字典形 / ない形 + つもりだ
詳細例句
夏休なつやすみは海うみへ行いくつもりです。
暑假打算去海邊。
今日きょうはテレビを見みないつもりです。
今天打算不看電視。
将来しょうらい、日本にほんで働はたらくいつもりです。
將來打算在日本工作。
16. ～ておく
用法：1.為了某個目的而事先做好準備。 2.讓狀態維持現狀。常譯為「事先...」。
接續：動詞て形 + おく
詳細例句
旅行りょこうの前に、ホテルを予約よやくしておきます。
旅行前，先預約好旅館。
客きゃくが來くるので、部屋へやを掃除そうじしておきました。
因為客人要來，所以事先打掃了房間。
還用だから、そのままにしておいてください。
因為還要用，請維持現狀放著。
17. ～てあげる / ～てもらう / ～てくれる
用法：表示動作的授受。「てあげる」我為他人做；「てもらう」請他人做；「てくれる」他人為我做。
接續：動詞て形 + あげる/もらう/くれる
詳細例句
友達ともだちに本ほんを貸かしてあげました。
我借書給朋友。
田中たなかさんに日本語にほんごを教おしえてもらいました。
我請田中先生教我日語。
母ははがシャツを買かってくれました。
媽媽買了襯衫給我。
18. ～させる（使役）
用法：表示指示、命令或允許某人做某事。常譯為「讓...做」、「叫...做」。
接續：動詞使役形
詳細例句
母ははは子供こどもに勉強べんきょうさせます。
媽媽叫小孩讀書。
先生せんせいは学生がくせいに本ほんを読よませました。
老師讓學生讀書。
彼かれにこの仕事をさせましょう。
讓他做這份工作吧。
19. ～られる（受身）
用法：表示被動，即動作的對象作為主詞。常譯為「被...」。
接續：動詞被動形
詳細例句
犬いぬに手てを噛かまれました。
手被狗咬了。
母ははに日記を見みられました。
日記被媽媽看了。
雨あめに降ふられて、ぬれてしまいました。
被雨淋了，全身都濕透了。
20. ～なさい
用法：父母或老師對孩子、學生下達輕微的命令。常譯為「請去...」、「去做...」。
接續：動詞ます形(去ます) + なさい
詳細例句
早はやく野菜やさいを食たべなさい。
趕快把蔬菜吃掉。
靜しずかにしなさい。
安靜一點。
もう寝ねなさい。
已經該睡覺了。
21. ～ために
用法：1.表示目的。 2.表示原因。常譯為「為了...」、「因為...」。
接續：動詞字典形 / 名詞+の + ために
詳細例句
留学りゅうがくのために、貯金ちょきんしています。
為了留學，正在存錢。
家いえを買かうために、一生懸命働はたらきます。
為了買房子，拼命工作。
事故じこのために、電車でんしゃが遅おくれています。
因為事故的緣故，電車遲延了。
22. ～ように
用法：1.表示目的。 2.表示請求、勸告。常譯為「為了能夠...」、「希望...」。
接續：動詞字典形 / ない形 + ように
詳細例句
忘わすれないように、メモします。
為了不忘記，記下筆記。
早はやく日本語にほんごが話はなせるように、練習れんしゅうします。
為了能夠早點說日語，練習著。
合格ごうかくできるように祈いのっています。
祈禱著能夠考試合格。
23. ～（さ）せられる（使役受身）
用法：表示被迫做某事。常譯為「不得不...」、「被迫...」。
接續：動詞使役被動形
詳細例句
母ははに部屋へやを掃除そうじさせられました。
被媽媽強迫打掃房間。
子供こどものとき、ピアノを習ならわせられました。
小時候被迫學習鋼琴。
昨日きのうはお酒を飲のませられました。
昨天被迫喝酒了。
24. ～すぎる
用法：表示動作或狀態超過了適當的程度。常譯為「太...」、「過於...」。
接續：動詞ます形(去ます) / 形容詞(去語尾) + すぎる
詳細例句
昨日きのうはお酒を飲のみすぎました。
昨天酒喝太多了。
この料理りょうりは辛からすぎます。
這道菜太辣了。
この問題もんだいは難むずかしすぎます。
這個題目太難了。
25. ～やすい / ～にくい
用法：表示做某事很容易或很困難。常譯為「容易...」、「難以...」。
接續：動詞ます形(去ます) + やすい / にくい
詳細例句
このペンはとても書かきやすいです。
這支筆很好寫。
この本ほんは難むずかしくて、讀よみにくいです。
這本書太難了，很難讀。
あの店みせは場所ばしょがわかりにくいです。
那家店的地點很難找。
26. ～だろう
用法：表示推測，口語為「～でしょう」。常譯為「大概是...吧」。
接續：普通形 / 名詞 / な形容詞 (去だ) + だろう
詳細例句
明日あしたは晴はれるだろう。
明天大概會放晴吧。
彼かれはもう知しっているだろう。
他大概已經知道了吧。
それは本当ほんとうだろうか。
那大概是真的吧（自問）。
27. ～がする
用法：用於感受到的味道、聲音、氣味等。常譯為「有...（味道/聲音/氣息）」。
接續：名詞 + がする
詳細例句
いい匂においがします。
有香味。
隣となりの部屋へやから変へんな音おとがします。
隔壁房間傳來奇怪的聲音。
このスープは変な味あじがします。
這湯有股怪味。
28. ～たばかり
用法：表示動作剛結束不久，主觀感受上的「剛...」。
接續：動詞た形 + ばかり
詳細例句
さっき晝ひるごはんを食たべたばかりです。
剛剛才吃過午飯。
日本にほんに來きたばかりなので、まだよくわかりません。
因為剛來日本，還不太清楚。
このカメラは買かったばかりなのに、もう壊こわれました。
這台相機才剛買，居然就壞了。
29. ～のに
用法：表示轉折，帶有意外、驚訝或不滿的情緒。常譯為「明明...卻...」。
接續：普通形 / 名詞+な / な形容詞+な + のに
詳細例句
薬くすりを飲のんだのに、熱ねつが下がりません。
明明吃了藥，燒還是沒退。
一生懸命いっしょうけんめい勉強べんきょうしたのに、不合格でした。
明明拼命讀書了，卻還是沒及格。
日曜日にちようびなのに、仕事しごとをしなければなりません。
明明是星期天，卻不得不工作。
30. ～ようだ
用法：1.比喻。 2.主觀感覺的推測。常譯為「好像...」。
接續：普通形 / 名詞+の / な形容詞+な + ようだ
詳細例句
あの人はまるでお人形さんのようです。
那個人簡直像娃娃一樣。
誰だれか来きたようです。
好像有人來了。
この料理りょうりは、ちょっと甘あまいようです。
這道菜好像有點甜。
31. ～みたいだ
用法：與「～ようだ」用法相似，但更口語化。常譯為「像...」、「好像...」。
接續：普通形 / 名詞 / な形容詞 (去だ) + みたいだ
詳細例句
あの人は日本人のみたいですね。
那個人好像日本人呢。
寶たからくじが当たったみたいです。
好像中樂透了。
お菓子の家みたいでかわいい。
像糖果屋一樣可愛。
32. ～らしい
用法：1.傳聞推斷（聽說）。 2.典型的樣貌（具有...的風格）。常譯為「據說好像...」、「具備...的特色」。
接續：普通形 / 名詞 / な形容詞 (去だ) + らしい
詳細例句
噂うわさによると、彼かれは結婚けっこんするらしいです。
據傳聞，他好像要結婚了。
今日きょうは春はるらしいいい天気てんきです。
今天是很有春天氣息的好天氣。
先生せんせいは明日お休やすみらしいです。
據說老師明天要請假。
33. ～てくる / ～ていく
用法：表示時間的流逝。一段時間持續到現在用「てくる」，從現在持續到未來用「ていく」。
接續：動詞て形 + くる / いく
詳細例句
ずっと日本語にほんごを勉強べんきょうしてきました。
一直以來都在學習日語（到現在）。
これからも頑張がんばっていきます。
今後也會繼續努力下去。
暖あたたかくなってきましたね。
天氣漸漸變暖了呢。
34. ～お～になる（尊敬）
用法：用於抬高對方的地位，表達敬意。常譯為「尊上的您（做某事）」。
接續：お + 動詞ます形(去ます) + になる
詳細例句
先生せんせいはもうお歸かえりになりました。
老師已經回去了。
社長しゃちょうは會議かいぎの資料しりょうをお讀よみになりました。
社長已經閱讀過會議資料了。
何を注文なさいますか。
請問您要點什麼？
35. ～お～する（謙譲）
用法：降低自己的地位，以對他人表示敬意。常譯為「敝人為您（做某事）」。
接續：お + 動詞ます形(去ます) + する
詳細例句
荷物にもつをお持もちします。
我來幫您提行李。
駅えきまでお送おくりします。
我送您去車站。
明日あした、お電話でんわします。
我明天會給您打電話。
36. ～はずがない
用法：表示強烈的否定推測，絕不可能。常譯為「絕不可能...」。
接續：普通形 / 名詞+の / な形容詞+な + はずがない
詳細例句
彼かれが嘘うそをつくはずがありません。
他絕不可能撒謊。
そんなことが起おこるはずがありません。
絕不可能發生那種事。
まだ食たべたばかりですから、お腹が空すくはずがありません。
因為才剛吃過，不可能肚子餓。
37. ～ばかり
用法：表示全部都是、光是。常譯為「淨是...」、「光是...」。
接續：名詞 / 動詞て形 + ばかり
詳細例句
ゲームばかりしないでください。
請不要光是玩遊戲。
肉にくばかり食べてはいけません。
不能光吃肉。
あの人はいつも文句もんくばかり言っています。
那個人老是在抱怨。
38. ～によって
用法：1.表示手段、方法。 2.表示原因。 3.表示被動句中的動作主體。
接續：名詞 + によって
詳細例句
バスによって駅えきへ行いきます。
搭公車去車站。
不注意ふちゅういによって、事故じこが起おこりました。
因為不注意而發生了事故。
電話はグラハム・ベルによって発明はつめいされました。
電話是由貝爾發明的。
39. ～中（ちゅう / じゅう）
用法：「ちゅう」表示動作正在進行；「じゅう」表示整個範圍或整個期間。
接續：名詞 + 中
詳細例句
今いま、電話でんわ中ちゅうです。
現在正在通話中。
一年いちねん中じゅう、暑あついです。
一年到頭都很熱。
世界せかい中じゅうを旅行りょこうしたい。
想去全世界旅行。
40. ～おかげで
用法：表示託某人或某事的福，得到了好的結果。常譯為「託...的福」、「多虧...」。
接續：普通形 / 名詞+の / な形容詞+な + お加之で
詳細例句
先生せんせいのおかげで、合格ごうかくできました。
多虧了老師，我考試合格了。
薬くすりを飲のんだおかげで、熱ねつが下がりました。

あなたが手伝てつだってくれたおかげで、助たすかりました。
多虧了你的幫忙，幫了我大忙。
41. ～し
用法：用於列舉多個原因或理由，有時暗示還有其他理由。常譯為「既...又...」、「而且...」。
接續：普通形 + し
詳細例句
この店は安いし、美味しいし、いつもいっぱいです。

明天既放假又沒事，我要好好睡一覺。
42. 使役形（～せる / ～させる）
用法：表示「讓某人做某事」或「使某人產生某種情感」。常譯為「叫...做」、「讓...做」。
接續：第一類動詞：語尾變為あ段 + せる；第二類動詞：去る + させる；第三類動詞：来る→來させる / する→させる
詳細例句
母は弟に部屋を掃除させました。
母親叫弟弟打掃房間。

テストでいい点を取って、両親を喜ばせたい。
想在考試中取得好成績，讓父母開心。
43. 受身形（～れる / ～られる）
用法：表示被動，即動作的主體受到某種動作的影響。常譯為「被...」。
接續：第一類動詞：語尾變為あ段 + れる；第二類動詞：去る + られる；第三類動詞：来る→來られる / する→される
詳細例句
私は犬に手をかまれました。

這座寺廟是500年前建造的。
44. 使役受身形（～せられる / ～させられる）
用法：表示「被迫做某事」，即原本不想做但被要求不得不做。常譯為「被迫...」。
接續：第一類動詞：語尾變為あ段 + せられる；第二類動詞：去る + させられる
詳細例句
子供の時、母に野菜を食べさせられました。
小時候，被迫吃蔬菜。
昨日は遅くまで残業させられました。
昨天被迫加班到很晚。
毎週、作文を書かされます。
每週都被迫寫作文。
45. ～なさい
用法：用於上級對下級下達指令，語氣較命令形溫和。常譯為「去...」、「請...」。
接續：動詞ます形(去ます) + なさい
詳細例句

已經很晚了，趕快去睡覺。
静かにしなさい。
請安靜。
宿題をやってしまいなさい。
把作業做完。
46. ～ために
用法：1. 表示目的（為了...）。 2. 表示原因（因為...）。
接續：動詞字典形 / 名詞 + の + ために
詳細例句
健康のために、毎日走っています。
為了健康，每天都在跑步。

為了在日本工作，正在學習日語。
事故のために、電車が止まっています。
因為事故，電車停駛了。
47. ～ように（目的）
用法：表示為了達到某種狀態或目標。後項多接非意志動詞或可能動詞。常譯為「為了能...」。
接續：動詞字典形 / ない形 / 可能形 + ように
詳細例句
忘れないように、メモをしておきます。
為了不忘記，我先記下筆記。
早く読めるように、たくさん練習します。
為了能讀得快，我要多練習。
病気が治るように、ゆっくり休んでください。
為了能讓病早點好，請好好休息。
48. ～ように（依頼）
用法：用於間接轉述命令、要求或建議。語氣較為委婉。常譯為「請...」、「要...」。
接續：動詞字典形 / ない形 + ように (言う/傳える/頼む)
詳細例句
先生に明日は休まないように言われました。
被老師叮嚀明天不要請假。
田中さんに私の部屋に来るように伝えてください。
請轉告田中先生來我的房間。
静かにするように注意されました。
被提醒要保持安靜。
49. ～ように（樣態）
用法：表示比喻或舉例，呈現出某種狀態或樣子。常譯為「像...一樣」。
接續：名詞 + の / 動詞普通形 + ように
詳細例句
彼は日本人のように日本語が上手です。
他日語講得就像日本人一樣好。
知っているように、日本は島国です。
正如大家所知，日本是個島國。
前に話したように、来月結婚します。
就像之前說過的，我下個月要結婚。
50. ～はずだ
用法：根據某種客觀事實或根據，進行有把握的推測。常譯為「應該會...」、「理所當然...」。
接續：普通形 / 名詞+の / な形容詞+な + はずだ
詳細例句
今日は日曜日だから、銀行は休みのはずです。
今天是星期日，銀行應該沒開。

他在日本待了10年，應該懂日語。
荷物は明日届くはずです。
包裹應該明天會寄到。
51. ～ば～ほど
用法：表示隨著前項程度的增加，後項的程度也隨之改變。常譯為「越...越...」。
接續：動詞假定形 + 動詞字典形 + ほど
詳細例句
日本語は勉強すればするほど難しくなります。
日語越學越覺得難。

這首歌越聽越喜歡。
荷物は軽ければ軽いほどいいです。
行李越輕越好。
52. ～ばかりでなく
用法：表示不僅僅是前項，還有後項。常譯為「不僅...而且...」。
接續：普通形 / 名詞 / な形容詞+な + ばかりでなく
詳細例句
彼は英語ばかりでなく、日本語も話せます。
他不僅會說英語，還會說日語。

あの人は歌が上手なばかりでなく、ピアノも弾けます。
那個人不僅唱歌好聽，還會彈鋼琴。
53. ～までに
用法：表示某個動作或狀態必須在某個時間點之前完成。常譯為「在...之前」。
接續：名詞 / 動詞字典形 + までに
詳細例句
明日の5時までに書類を出してください。
請在明天5點前提交文件。
会議が始まるまでに、準備をしておきます。
在會議開始之前，先做好準備。
冬になるまでに、コートを買いたいです。
想在入冬前買一件大衣。
54. ～まま
用法：表示維持原本的狀態不變，而進行下一個動作。常譯為「就那樣...」、「照舊...」。
接續：動詞た形 / ない形 / 名詞+の / な形容詞+な / い形容詞 + まま
詳細例句
窓を開けたまま、寝てしまいました。
就那樣開著窗戶睡著了。
眼鏡をかけたまま、顔を洗ってしまいました。
就那樣戴著眼鏡洗了臉。
この辺りは昔のままの景色が残っています。
這一帶保留著和以前一樣的景色。
55. ～みたいだ
用法：1. 表示比喻。 2. 表示推測。語氣較為口語。常譯為「像...一樣」、「好像...」。
接續：普通形 / 名詞 / な形容詞 (去だ) + みたいだ
詳細例句
宝くじに当たったなんて、夢みたいです。
竟然中了彩券，就像做夢一樣。
外は雨が降っているみたいです。
外面好像在下雨。
彼女は人形みたいに可愛いです。
她長得像洋娃娃一樣可愛。
56. ～向きだ
用法：表示適合、相配，非刻意設計但剛好合適。常譯為「適合...」。
接續：名詞 + 向きだ
詳細例句

57. ～向けだ
用法：表示以某對象為目標，刻意設計或製作的。常譯為「針對...」、「面向...」。
接續：名詞 + 向けだ
詳細例句

58. ～もし～たら
用法：強調假設定的情況，「もし」起到加強假設語氣的作用。常譯為「萬一...」、「如果...」。
接續：もし + 動詞た形 + ら
詳細例句
もし1億円あったら、家を買いたいです。

もし道に迷ったら、電話してください。
如果你迷路了，請給我打電話。
59. ～ものだ
用法：表示理所當然的事實、社會常識，或是回憶過去的習慣。常譯為「應當...」、「總是...」。
接續：動詞字典形 / ない形 / い形 / な形+な + ものだ
詳細例句
約束は守るものです。
約定就是要遵守的。
子供のころは、よくこの公園で遊んだものです。
小時候，經常在這個公園玩耍。
時が過ぎるのは早いものですね。
時間過得真快啊。
60. ～ものか
用法：強烈地否定某件事，或是表達絕不那樣做的決心。常譯為「絕不...」、「哪會...」。
接續：普通形 / 名詞+な / な形容詞+な + ものか
詳細例句

那種店，我絕不去第二次。
負けるものか。
絕不能輸。
彼に本当のことが言えるものか。
哪能對他說真話呢。
61. ～ものなら
用法：表示「如果可以實現（實際上很難）的話」，常用於表達願望。常譯為「如果能...的話」。
接續：動詞可能形 + ものなら
詳細例句
帰れるものなら、今すぐ国へ帰りたい。
如果能回去的話，現在就想回國。
やり直せるものなら、やり直したい。
如果能重來的話，我想重頭來過。
私の代わりができるものなら、やってみてください。
如果能取代我的話，你就試試看。
62. ～ものの
用法：雖然認可前項的事實，但結果卻不如預期，或是與前項相反。常譯為「雖然...但是...」。
接續：普通形 / 名詞+である / な形容詞+な / である + ものの
詳細例句
車を買ったものの、ほとんど乗っていません。
雖然買了車，但幾乎沒在開。
給料は高いものの、仕事がとても大変です。
薪水雖然高，但工作非常辛苦。
謝ったものの、許してもらえなかった。
雖然道歉了，但沒得到原諒。
63. 意向形（～う / ～よう）
用法：用於表示自己的意志、打算，或是邀約、提議。常譯為「打算...」、「讓我們...吧」。
接續：第一類動詞：語尾變為お段 + う；第二類動詞：去る + よう；第三類動詞：来る→來よう / する→しよう
詳細例句
疲れたから、少し休もう。
累了，休息一下吧。
将来、日本で働こうと思っています。
打算將來在日本工作。
今日はもう帰ろう。
今天就回家吧。
64. ～ようとする
用法：表示正要開始做某動作，或是嘗試要做某事。常譯為「正打算...」、「企圖...」。
接續：動詞意向形 + とする
詳細例句
出かけようとしたとき、電話が鳴りました。
正打算出門時，電話響了。
電車に乗ろうとしたが、ドアが閉まってしまった。
正想上電車時，車門卻關上了。
犬があなたの靴を噛もうとしています。
狗狗正想咬你的鞋子。
65. ～わけだ
用法：表示對某件事的結論或理由感到理所當然，或是換句話說的結論。常譯為「難怪...」、「原來是...」。
接續：普通形 / 名詞+な / である / な形容詞+な + わけだ
詳細例句
外は2度ですか。寒いわけですね。
室外才2度嗎？難怪這麼冷。
1500円の3割引だから、1050円なわけだ。
1500日圓打七折，所以是1050日圓。
彼の母親は日本人だから、彼はハーフなわけだ。
他的母親是日本人，所以他就是混血兒了。
66. ～わけがない
用法：表示從客觀理由來看，絕不可能發生某事。語氣比「はずがない」更強。常譯為「絕不可能...」、「沒有道理...」。
接續：普通形 / 名詞+の / な形容詞+な + わけがない
詳細例句
そんな難しい問題、私にできるわけがありません。
那麼難的問題，我絕不可能會做。
彼が犯人なわけがありません。
他絕不可能是犯人。
昨日買ったばかりだから、壊れるわけがない。
因為昨天才剛買，不可能會壞掉。
67. ～わけではない
用法：部分否定某種情況，表示事實並不完全像對方所想的那樣。常譯為「並非...」、「不代表...」。
接續：普通形 / 名詞+の / である / な形容詞+な + わけではない
詳細例句
嫌いなわけではありませんが、お肉はあまり食べません。
並不是討厭，只是不太吃肉。
金持ちがいつも幸せなわけではない。
有錢人並不一定總是幸福的。
全く準備をしていないわけではない。
並不是完全沒有做準備。
68. ～わけにはいかない
用法：表示受到社會常識、道德或個人良心的制約，而不能做某事。常譯為「不能...」。
接續：動詞字典形 + わけにはいかない
詳細例句
明日テストがあるので、遊んでいるわけにはいきません。
明天有考試，所以我不能只顧著玩。
大事な会議があるから、休むわけにはいかない。
因為有重要的會議，所以我不能請假。
親友を裏切るわけにはいきません。
我不能背叛好友。
69. ～をきっかけに
用法：以某事為契機或開端，引發了後續的變化或新行動。常譯為「以...為契機」、「趁著...」。
接續：名詞 + をきっかけに
詳細例句
アニメをきっかけに、日本語に興味を持ちました。
以動漫為契機，我對日語產生了興趣。

怪我をきっかけに、スポーツを辞めました。
以受傷為契機，我放棄了體育。
70. ～を中心に
用法：表示以某人 or 某事作為重心、核心來進行動作。常譯為「以...為中心」。
接續：名詞 + を中心に
詳細例句
駅を中心に、新しい町が作られています。
以車站為中心，正在建造新市鎮。
地球は太陽を中心に回っています。
地球繞著太陽旋轉。
この学校は、会話を中心に教えています。
這間學校主要以會話教學為中心。
71. ～を問わず
用法：表示無論在任何情況下、不論對象為何，結果都一樣。常譯為「不論...」、「不分...」。
接續：名詞 + を問わず
詳細例句
このスポーツは年齢を問わず、誰でも楽しめます。
這項運動不分年齡，誰都能享受。

不分國籍，廣泛招募人才。
昼夜を問わず、工事が進められています。
不分晝夜，工程都在進行中。
72. ～をはじめ
用法：舉出一個代表性的例子，暗示還有其他類似的事項。常譯為「以...為首」、「以及...」。
接續：名詞 + をはじめ
詳細例句
富士山をはじめ、日本には多くの美しい山があります。
以富士山為首，日本有許多美麗的山。

以社長為首，全體員工都出席了會議。
ひらがなをはじめ、漢字も勉強しなければなりません。
除了平假名之外，還必須學習漢字。
73. ～をもとに
用法：以某事為基準、依據、素材來創作出新的東西。常譯為「基於...」、「根據...」。
接續：名詞 + をもとに
詳細例句

試験の結果をもとに、クラスを分けます。
根據考試結果來分班。
この小説をもとに、ドラマが制作された。
基於這部小說製作了電視劇。
74. ～ということだ
用法：1. 轉述聽到的內容（聽說...）。 2. 用於對前面的內容進行解釋說明（也就是說...）。
接續：普通形 + ということだ
詳細例句
天気予報によると、明日は晴れるということです。
根據天氣預報，聽說明天會放晴。
彼が来られないということは、私たちがやらなければならないということです。
他不能來這件事，就表示我們必須去做。
合格したということは、努力が報われたということだ。
考試合格了，就代表努力得到了回報。
75. ～といっても
用法：補充說明前述內容與實際情況的差距。常譯為「雖說...但是...」、「話雖如此...」。
接續：普通形 / 名詞 + といっても
詳細例句
料理ができるといっても、卵焼きぐらいです。
雖說會做菜，但也只是會做煎蛋捲而已。
家を買ったといっても、とても小さい中古の家です。
雖然說買了房子，但也是非常小的二手房。
忙しいといっても、週末は休めます。
話雖說很忙，但週末還是可以休息的。
76. ～としたら

90. ～によって / ～により / ～による / ～によっては

92. ～を中心に / ～を中心として

113. ～につれて / ～につれ
用法：表示隨著一方的變化，另一方也同時發生變化。多用於自然或必然的變化。常譯為「隨著...」。
接續：名詞 / 動詞字典形 + につれて
詳細例句
時間が経つにつれて、悲しみが薄れてきた。
隨著時間的流逝，悲傷漸漸淡去了。
町の発展につれて、緑が少なくなった。
隨著城鎮的發展，綠地減少了。
年をとるにつれ、忘れっぽくなる。
隨著年齡增長，變得健忘。
114. ～にとって（は） / ～にとっても / ～にとっての
用法：表示對某人而言。從該立場進行評價或判斷。常譯為「對...來說」。
接續：名詞 + にとって
詳細例句
私にとって、家族は一番大切です。
對我來說，家人是最重要的。
環境問題は人類にとって大きな課題だ。
環境問題對人類來說是個重大課題。
犬にとっても、散歩は必要だ。
對狗來說，散步也是必要的。
115. ～に伴って / ～に伴い / ～に伴う
用法：表示隨著某個現象的發生，同時引發了另一個現象。語氣較正式。常譯為「伴隨著...」。
接續：名詞 / 動詞字典形 + に伴って
詳細例句
人口の増加に伴って、様々な問題が生じた。
伴隨著人口增加，產生了各種問題。
経済発展に伴い、生活が豊かになった。
伴隨著經濟發展，生活變富裕了。
手術に伴う危険性について説明を受けた。
聽取了關於伴隨手術的危險性的說明。
116. ～に反して / ～に反し / ～に反する / ～に反した
用法：表示與規則、期待或預測相反。常譯為「與...相反」、「違反...」。
接續：名詞 + に反して
詳細例句
親の期待に反して、彼は学校を辞めた。
違反了父母的期待，他退學了。
天気予報に反して、雨が降った。
與天氣預報相反，下雨了。
ルールに反する行為。
違反規則的行為。
117. ～にわたって / ～にわたり / ～にわたる / ～にわたった
用法：表示在時間的整個長度或空間的整個範圍內。常譯為「歷經...」、「橫跨...」。
接續：名詞 + にわたって
詳細例句
会議は3時間にわたって行われた。
會議歷經三小時才結束。
関東地方全域にわたって雨が降っている。
降雨橫跨了整個關東地區。
長期にわたる研究。
歷經長期的研究。
118. ～ば～ほど
用法：表示某事物的程度越高，另一事物的程度也隨之提高。常譯為「越...越...」。
接續：動詞假定形 + 字典形 + ほど
詳細例句
日本語は勉強すればするほど面白い。
日語越學越有趣。
早ければ早いほどいいです。
越早越好。
考えれば考えるほどわからなくなる。
越想越不懂。
119. ～ばかりか / ～ばかりでなく
用法：表示不僅是前項，還有更進一步的後項。常譯為「不僅...而且...」。
接續：普通形 / 名詞 / な形容詞+な + ばかりか
詳細例句
彼は英語ばかりか、フランス語も話せる。
他不僅會說英語，連法語也會說。
この薬は効かないばかりか、副作用もある。
這藥不僅無效，而且還有副作用。
彼女は美しいばかりでなく、性格もいい。
她不僅漂亮，而且性格也很好。
120. ～わけにはいかない
用法：表示受到社會常識、道德、責任或心理等因素的制約，而不能做某事。常譯為「不能...」、「無法...」。
接續：動詞字典形 + わけにはいかない
詳細例句
大事だいじな会議かいぎがあるから、休やすむわけにはいかない。

親友しんゆうを裏切うらぎるわけにはいきません。
